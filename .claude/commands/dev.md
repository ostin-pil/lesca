# Run Development Mode

Start the CLI in development mode with watch.

## Arguments

- `$ARGUMENTS` - Required: CLI command and arguments (e.g., `scrape two-sum` or `scrape-list --difficulty Easy`)

## Steps

1. Run in dev mode: `npm run dev -- $ARGUMENTS`
2. Watch for and report any errors
3. Show the output/results

## Common Commands

- `scrape <slug>` - Scrape a specific problem
- `scrape-list` - Scrape problems from a list
- `config show` - Show current configuration
- `auth` - Run authentication flow
